Эта инструкция для админа.
Во-первых Вы являетесь админом.
Во-вторых Вам доступен следующий функционал:
-просмотреть список серверов;
-просмотреть статус серверов;
-просмотреть статус всех устройств от удаленных серверов;
-просмотреть список пользователей;
-просмотреть список админов.
В-третьих Вы можете с помощью набора КОМАНД добавить, удалить из списков админов (если Вы обладаете правами супер-админа), пользователей, серверов.
Команды:
ADD USER - добавляет пользователя, затем введите строго соблюдая последовательность [ID][пробел][Имя и Фамилия]
DELETE USER - удаляет пользователя, затем введите строго соблюдая последовательность [ID][пробел][Имя и Фамилия] 
ADD ADMIN - добавляет админа, затем введите строго соблюдая последовательность [ID][пробел][@логин]
DELETE ADMIN - удаляет админа, затем введите строго соблюдая последовательность [ID][пробел][@логин]
ADD SERVER - добавляет сервер, затем введите строго соблюдая последовательность [IP][пробел][название сервера]
DELETE SERVER - удаляет сервер, затем введите строго соблюдая последовательность [IP][пробел][название сервера]
